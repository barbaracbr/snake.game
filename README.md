## 🎮 Como jogar

- **Setas do teclado** controlam a direção da cobrinha:
  - ⬅️ Esquerda
  - ⬆️ Cima
  - ➡️ Direita
  - ⬇️ Baixo
- Coma as 🍎 **maçãs vermelhas** para crescer e ganhar pontos.
- Desvie das 💣 **bombas pretas** — se encostar, é **game over**!
- Não bata nas bordas ou em si mesma.
